# Условное ветвление: if, '?'

## if() {}

```js
let condition = true;
if (condition) {
    // если condition = true
} else if (condition > 1) {
    // если condition > 1
} else {
    // если condition = false
}
```

## Условный оператор „?“

```js
let result = condition ? значение1 : значение2;

```

переписать на '?'

let message;

if (login == 'Сотрудник') {
  message = 'Привет';
} else if (login == 'Директор') {
  message = 'Здравствуйте';
} else if (login == '') {
  message = 'Нет логина';

  message = '';
}
